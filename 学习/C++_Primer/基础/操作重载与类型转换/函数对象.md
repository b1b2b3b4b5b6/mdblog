
# 函数对象
如果类定义了调用运算符`()`,则该类的对象称作**函数对象**




## 标准库function表
![](https://raw.githubusercontent.com/b1b2b3b4b5b6/pic/main/PicGo/202110221532544.png)

用法

```C++
map<string function<int(int,int)>> binops = {
    {"+", add},                                 //函数指针
    {"-", std::minus<int>()},                   //标准库函数对象
    {"/", divide},                              //用户定义的函数对象
    {"*", [](int i, int j){return i*j;}},       //未命名的lambda
    {"%", mod}                                  //命名了的lambda对象
};
```

不能将重载函数的名字存入function类型的对象中,应存储函数指针而非函数名字

```C++
int (*fp)(int, int) = add; //指针指向的add是接受两个int的版本
binops.insert({"+", fp}); //正确
```
