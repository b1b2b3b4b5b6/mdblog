# 链接
# 一览
|    链接参数     |           举例            |             说明              |
| --------------- | ------------------------ | ----------------------------- |
| -l              | -lmy                     | 链接libmy.so                  |
| -L              | -L/usr/bin               | 链接搜索目录/usr/bin           |
| -L              | -L /usr/bin/my.so        | 链接/usr/bin/my.so            |
| -WL,-rpath      | -WL,-rpath,/usr/bin      | 只在运行时链接搜索目录/usr/bin |
| -WL,-rpath-link | -WL,-rpath-link,/usr/bin | 只在编译时链接搜索目录/usr/bin |

**注意项**
- `-l`会自动前缀添加`lib`,后缀添加`.so`,在连接目录搜索动态库
- cmake使用`target_link_libraries`,即使写全路径lib,但实际链接时可能会被简化为`-L -l`使用的形式, 需要用`target_link_options`直接写链接参数
