# 时间同步工具
|   程序   |                                  说明                                   |
| ------- | ----------------------------------------------------------------------- |
| ptp4l   | 负责在网络中同步时钟（基于IEEE 1588 PTP协议）, 是Linux内核PTP实现的一部分 |
| phc2sys | 将硬件时钟（PHC 就是网卡时间）同步到系统时钟（或反之），提高时间一致性     |
| pmc     | 与 ptp4l 守护进程通信，读取状态（比如主时钟是谁）。                       |
| hwclock | 管理本地硬件RTC，但在PTP场景下作用有限。                                  |
| ptpd    | 类似ptp4l, 但是不属于linux内核实现, 配置灵活，功能丰富                    |
