# 编译

## 一览
| 步骤名 |     命令     | 等价命令 |  输出文件  |
| ------ | ----------- | -------- | ---------- |
| 预处理 | cpp         | gcc -E   | .i .ii     |
| 编译   | cc1 cc1plus | gcc -S   | .s         |
| 汇编   | as          | gcc -c   | .o .obj    |
| 链接   | ld          | gcc      | 可执行文件 |
| 四合一 | gcc         |          | 可执行文件 |
