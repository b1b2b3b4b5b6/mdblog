# 杂记

##### 指定字面值的类型
字符和字面串子面值
| 前缀 | 含义                       | 类型     |
| :-- | :------------------------ | :------- |
| u    | Unicode 6字符              | char16_t |
| U    | Unicode 6字符              | char32_t |
| L    | 宽字符                     | wchar_t  |
| u8   | UTF8（仅用于字符串子面常量） | char     |

整型字面值
| 后缀     | 最小匹配类型 |
| :------- | :---------- |
| u 或 U   | unsigned    |
| l 或 L   | long        |
| ll 或 LL | long long   |

浮点型字面值
| 后缀   | 类型        |
| :----- | :---------- |
| f 或 F | float    |
| l 或 L | long double |

##### 琐碎
将指针赋值为空时，尽量使用nullptr
