# 类型转换运算符
是类的一种**特殊成员函数**,负责将一个类类型的值转换成其他类型,形式如下
```C++
operator type() const;
```
* 只要type能作为函数的返回类型，就可以使用类型转换运算符定义
* 类型转换函数必须是类的成员函数, 不能声明返回类型，形参列表必须为空

> 尽管编译器只能执行一个用户定义的类型转换，但是隐式的用户定义类型转换可以置于一个标准类型转换之前或之后

## 类类型的转换
由转换构造函数和类型转换运算符共同定义,也被称作用户定义的类型转换

## 显式的类型转换运算符

```C++
explicit operator int() const { return val; } 
```

使用explicit编译器将不会进行隐式转换，除非以下情况
* `if while do`语句的条件部分
* `for`语句的条件表达式
* `! || &&`逻辑运算符
* `?:`条件运算符

## 二义性问题
存在多种转换可能时, 就可能会出现二义性问题, 只能显式调用解决
如果存在标准转换, 则标准类型转换将决定最佳匹配
