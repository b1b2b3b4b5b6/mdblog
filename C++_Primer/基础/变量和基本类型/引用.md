# 引用

- 引用是对象的别名
- 引用不是对象,故引用无法绑定到引用
- 引用必须被初始化(无法重新绑定到另外一个对象)
- 分为左值引用和右值引用

## 左值引用

* 绑定到左值上
* const引用可以绑定到右值
* 
```C++
int i = 42;
int &r = i;        
const int &r2 = i*42; //正确,即使i*42是右值
```

## 右值引用
* 绑定到右值上
    * 返回非引用的函数
    * 算数,关系,位,后置递增/递减运算符

## move
用法如下

```C++
int i = 0;
int &&r = std::move(i);
```

* 可以显式的将左值转换为对应的右值引用类型
* move应使用std::move,避免潜在冲突
